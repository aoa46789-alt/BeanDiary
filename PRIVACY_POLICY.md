# BeanDiary 개인정보 처리방침

## 요약

BeanDiary는 커피 기록을 위한 개인용 iOS 앱입니다. **별도의 회원가입·서버·계정이 없으며**, 대부분의 데이터는 사용자 기기에만 저장됩니다.

## 수집·저장하는 데이터

| 데이터 | 저장 위치 | 용도 |
|--------|-----------|------|
| 커피 기록, 원두, 시음 메모 | 기기 (SwiftData) | 앱 핵심 기능 |
| 사진·영상 첨부 | 기기 (앱 샌드박스) | 기록 첨부 |
| 카페 방문 정보·메모 | 기기 (SwiftData) | 지도 기능 |
| API 키 (Gemini, YouTube) | 기기 (UserDefaults) | 사용자가 입력한 AI·YouTube 기능 |
| 그라인더 설정 | 기기 (UserDefaults) | 레시피 환산 |

개발자는 위 데이터를 **자체 서버로 수집·저장하지 않습니다.**

## 제3자 서비스

사용자가 API 키를 설정한 경우, 앱은 다음 서비스에 **직접** 요청을 보냅니다.

- **Google Gemini API** — 원두 분석, 레시피 파싱, 카페 미리보기
- **YouTube Data API** — 레시피 영상 검색
- **Apple MapKit** — 카페 검색·지도 표시

각 서비스의 데이터 처리는 해당 제공자의 정책을 따릅니다.

- [Google Privacy Policy](https://policies.google.com/privacy)
- [Apple Privacy Policy](https://www.apple.com/legal/privacy/)

## 기기 권한

| 권한 | 목적 |
|------|------|
| 사진 라이브러리 | 커피 기록에 사진·영상 첨부 |
| 카메라 | 기록용 사진 촬영 |
| 위치 (사용 중) | 주변 카페 검색·지도 표시 |
| Live Activity | 브루잉 타이머 잠금화면 표시 |

권한은 해당 기능 사용 시에만 요청되며, 거부해도 기본 기록 기능은 사용할 수 있습니다.

## 데이터 삭제

앱을 삭제하면 기기에 저장된 모든 BeanDiary 데이터가 함께 삭제됩니다.

## 아동

BeanDiary는 13세 미만 아동을 대상으로 하지 않습니다.

## 변경

정책이 변경되면 이 문서와 앱 내 링크를 업데이트합니다.
